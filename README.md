# 상권분석 시스템

# 주제를 선택한 이유
2020년은 다사다난한 해입니다. 코로나와 경제침체로 인해 많은 소상공인들이 고통받고 있습니다.  
저희 팀은 프로젝트를 하더라도 사회에 조금이라도 기여할 수 있는 프로젝트를 주제로 하고 싶어했고,
현재 존재하는 상권분석 시스템의 속도가 매우 느렸기에 조금 더 빠른 속의 상권분석 시스템을 구현하고 싶었습니다. 
몇 백만개의 데이터를 csv 파일로 데이터베이스에 import 하여 그 데이터를 전처리할 수 있는 경험 할 수 있는 주제를 원했습니다.
이러한 욕구를 충족시킬 수 있는 주제가 상권분석이였습니다.
